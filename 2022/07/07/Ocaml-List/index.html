自己实现一下Ocaml的List库中的函数

val length : 'a list -> int 返回列表长度

val hd : 'a list -> 'a 返回首元素

val tl : 'a list -> 'a list 返回除去首元素后的剩余列表

val nth : 'a list -> int -> 'a 返回第n个位置的元素(第一个元素在位置0)

val append : 'a list -> 'a list -> 'a list 合并列表

val rev_append : 'a list -> 'a list -> 'a list 第一个列表翻转后与第二个列表合并

val rev : 'a list -> 'a list 翻转列表

val init : int -> (int -> 'a) -> 'a list 构造[f(0),f(1),f(2)...f(len)]

val concat : 'a list list -> 'a list 将多个列表合并
